# Running Auto-Notes Architect in Google Colab

To run the pipeline and Streamlit UI in Google Colab, follow these steps:

1. **Open a new Colab Notebook.**
2. **Install system dependencies:**
   ```bash
   !apt-get install -y ffmpeg
   ```
3. **Install Python dependencies:**
   ```bash
   !pip install streamlit openai-whisper spacy transformers openai reportlab
   !python -m spacy download en_core_web_sm
   ```
4. **Create the Project Structure:**
   Upload the `pipeline/`, `utils/`, and `app.py` files to your Colab environment.
5. **Run the Streamlit App:**
   ```bash
   !npm install -g localtunnel
   !streamlit run app.py & npx localtunnel --port 8501
   ```
6. **Access the UI:**
   Click the link generated by localtunnel to open the Auto-Notes Architect.

*Note: This React/TS version is optimized for web deployment. The Colab version requires the Python files provided in this project.*
